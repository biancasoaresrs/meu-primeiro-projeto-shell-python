# Projeto Shell Script + Python

Este projeto executa um script Shell que chama um código Python para realizar cálculos.

## Arquivos do projeto
* calculadora.sh → Script executável
* media.py → Código Python

---
## Como executar o arquivo .sh
1. Abrindo o terminal Linux

2. Dando permissão de execução:
chmod +x calculadora.sh

3. Executando o script:
./calculadora.sh
---

## Explicação do código Python:
O arquivo `media.py`:

* Lê dados de um arquivo CSV
* Separa as informações
* Calcula a média dos valores
* Mostra o resultado na tela

Exemplo de saída:
25.0
